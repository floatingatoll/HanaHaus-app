# HanaHaus

This repo consists of the HanaHaus website, hosted via GitHub Pages.

HanaHaus is a static website generated with jekyll. We are using
[jekyll-assets manager](https://github.com/ixti/jekyll-assets) to
handle local assets.

## Installation

```
bundle install
```

## Local development

```
rake serve
```

## Publish (to gh-pages)

```
rake publish
```
